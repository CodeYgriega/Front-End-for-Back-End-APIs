Interfaz gráfica e interactiva para APIs de Back-End

  • Aplicación web desarrolla con las tecnologías Angular y Bootstrap.
  • El objetivo de la aplicación es que, dinámicamente, sea efectiva para probar la API que el usuario le pase, visualizándola en estilo CRUD, donde se mostrará un listado con botones para:
    • Agregar
    • Editar
    • Eliminar
  • El objetivo se logra mediante la información básica que debe brindar el usuario sobre su API, como por ejemplo:
    • Las URLS para efectuar las distintas peticiones (GET, POST, PUT, DELETE).
    • Información sobre las propiedades a acceder en la respuesta obtenida de la API.
    • Propiedades en específico para mostrar correctamente cada elemento en el listado:
      • Título, nombre o propiedad elegida para mostrar en la card
      • Id para identificar el elemento y enviarlo por parámetro en las peticiones PUT y DELETE.
